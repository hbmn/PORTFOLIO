WORKS

無農薬・オーガニック野菜の定期便
架空ブランド「すぐ食べ」LP

WebSite
フロント技術 HTML / CSS / jQuery
命名規則 BEM
使用プラグイン -
使用ソフト Photoshop
対応端末 Mobile / Tablet / PC
担当範囲 ワイヤーフレーム、デザイン、バナー、コーディング

企画内容

背景

  • スクールの中間課題作品、架空ブランドのLPです。下記のコンセプトを元に作成しました。

目的

  • Webサイトは存在するが、お試し購入へ導くためのLPを作成。
  • 産地直送、新鮮な無農薬野菜を訴求する。

ターゲット

  • 成長期の子供がいるママ。
  • 働き盛りの夫にもまだまだ元気でいてほしいと思っている妻。
  • 健康志向、食生活にこだわっている方。
  • 食にお金を掛けられ、なるべく安心安全な物を食べたい方。

コンセプト

  • 新鮮さ、信頼感、安心安全、ナチュラルで明るい感じ

デザインについて

セットが届いてからどんな料理ができるのかをユーザーに想像させ、購入に繋げる狙いでお試しセットの内容例や料理レシピの紹介をオプションで組み込みました。

メインビジュアル下のバナーは、このサイトがどんなキャンペーンをしているのかを一目見て分かりやすくし、そのまま購入ページへ進めるようにと設置しました。バナーは自作しました。

「ご注文はこちら」ボタンはユーザーが押しやすく、ページ内で目立つボタンになるよう心掛けました。

コーディングについて

ファーストビューの「こだわり農家の朝どれ〜」タイトルはvwで実装し、閲覧するデバイスごとに最適なサイズになるようレスポンシブ実装をしました。

画面幅が狭いデバイスでも前後のテキストに重ならないよう調整しながら野菜の写真を配置しました。(下記画像:画面幅820px)

コメント

課題としての初めての作品でした。どうやってユーザーをコンバージョンへ導くかを考慮しユーザーの気持ちになって客観的に作成しました。 LPはユーザーへの説得や共感をデザインや構成で表現することで全体にストーリー性が生まれるところが普通のWebサイトと違って面白く、奥深く感じました。
