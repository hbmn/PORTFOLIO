HTML5

BEMを意識した命名規則で適切なマークアップを行い、インデント揃えやコメントアウトで分かりやすく丁寧なコーディングを心掛けております。 アクセシビリティの向上も意識し、誰もが利用しやすいサイト作りを目指します。

CSS3

メディアクエリでのレスポンシブ実装が可能です。vwやclampを使用しブレークポイント付近で差が無いように意識しています。 効率のよいCSS記述ができるよう、Sassの勉強も行いたいと思っております。

jQuery

jQeryではハンバーガーメニュー、モーダル、アコーディオンなどの簡単な記述とライブラリーのプラグインが可能です。

GitHub

ターミナルでのGitHub操作が可能です。GitFlowの流れを簡単に理解しています。

Photoshop

Photoshopでは主にデザインカンプの作成をします。webサイト用に写真レタッチ・加工・合成をして書き出しを行います。

Xd

Xdではデザインカンプ・ワイヤーフレームの作成をします。プロトタイプ、リンク共有など基本操作が可能です。

Lightroom

Webサイトで使用する写真素材のレタッチに使用しています。現在は趣味の一環として操作するレベルです。

Illustrator
